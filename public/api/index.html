API to access stored data

Data are stored daily in .csv format. The data are in JSON formt and have the following syntax:

{
  id:" ",
  start_timestamp:" ",
  stop_timestamp:" ",
  data: [gps0:"lat, long", acceleration0:"x, y, z", ...., gpsN:"lat, long", accelerationN:"x, y, z"]
}

To download the data use http get to http://ec2-13-58-154-188.us-east-2.compute.amazonaws.com:3335/api/data/[nome_file.csv]
